channelrequest.js
Definition of the ChannelRequest class. The ChannelRequest object encapsulates the logic for making a single request, either for the forward channel, back channel, or test channel, to the server. It contains the logic for the three types of transports we use in the BrowserChannel: XMLHTTP, Trident ActiveX (ie only), and Image request. It provides retry and timeout detection. This class is part of the BrowserChannel implementation and is not for use by normal application code.
