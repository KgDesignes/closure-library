goog.ui.CccButton(opt_class, opt_noPoundSignInHref)

Parameters

opt_class :
string=
Optional class for component. Default: 'goog-cccbutton'.
opt_noPoundSignInHref :
boolean=
An optional argument that, when true, instructs the button to not use the href="#" to create the link. The side effect of this option being true is that clicking the button triggers the window's onunload or onbeforeunload handler in IE. The upside of setting this to true is that the page will not scroll when clicking this button in IE.
