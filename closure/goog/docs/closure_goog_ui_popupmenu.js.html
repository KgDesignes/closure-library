popupmenu.js
A menu class for showing popups. A single popup can be attached to multiple anchor points. The menu will try to reposition itself if it goes outside the viewport. Decoration is the same as goog.ui.Menu except that the outer DIV can have a 'for' property, which is the ID of the element which triggers the popup. Decorate Example:
