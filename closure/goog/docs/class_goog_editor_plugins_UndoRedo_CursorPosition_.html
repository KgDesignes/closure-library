editor.plugins.UndoRedo.CursorPosition_ Extends Object
Stores the state of the selection in a way the survives DOM modifications that don't modify the user-interactable content (e.g. making something bold vs. typing a character). TODO: Completely get rid of this and use goog.dom.SavedCaretRange.
