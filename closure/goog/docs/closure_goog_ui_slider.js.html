slider.js
A slider implementation that allows to select a value within a range by dragging a thumb. The selected value is exposed through getValue(). To decorate, the slider should be bound to an element with the class name 'goog-slider-[vertical / horizontal]' containing a child with the classname 'goog-slider-thumb'. Decorate Example:
