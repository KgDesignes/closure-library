actionhandler.js
This file contains a class to provide a unified mechanism for CLICK and enter KEYDOWN events. This provides better accessibility by providing the given functionality to a keyboard user which is otherwise would be available only via a mouse click. If there is an existing CLICK listener or planning to be added as below - this.eventHandler_.listen(el, CLICK, this.onClick_); it can be replaced with an ACTION listener as follows: this.eventHandler_.listen( new goog.events.ActionHandler(el), ACTION, this.onAction_);
