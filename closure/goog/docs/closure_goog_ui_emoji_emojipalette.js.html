emojipalette.js
Emoji Palette implementation. This provides a UI widget for choosing an emoji from a palette of possible choices. EmojiPalettes are contained within EmojiPickers. See ../demos/popupemojipicker.html for an example of how to instantiate an emoji picker. Based on goog.ui.ColorPicker (colorpicker.js).
