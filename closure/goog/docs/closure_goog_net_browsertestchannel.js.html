browsertestchannel.js
Definition of the BrowserTestChannel class. A BrowserTestChannel is used during the first part of channel negotiation with the server to create the channel. It helps us determine whether we're behind a buffering proxy. It also runs the logic to see if the channel has been blocked by a network administrator. This class is part of the BrowserChannel implementation and is not for use by normal application code.
