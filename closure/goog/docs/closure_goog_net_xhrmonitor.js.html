xhrmonitor.js
Class used by XHR wrappers to publish their state to IframeIo or other components that need to know if any XmlHttpRequests are active. IframeIo needs to know if any XmlHttpRequests have been initiated from the result of an incremental data response, so that it can delay the destruction of the iframe. Otherwise firefox will error since the source context no longer exists. This class means that IframeIo does not have an explicit dependency on XhrIo. See https://bugzilla.mozilla.org/show_bug.cgi?id=369939 for a description of the problem and a minimal test case. This class's methods are no-ops for non-Gecko browsers.
