bezier.js
Represents a cubic Bezier curve. Uses the deCasteljau algorithm to compute points on the curve. http://en.wikipedia.org/wiki/De_Casteljau's_algorithm Currently it uses an unrolled version of the algorithm for speed. Eventually it may be useful to use the loop form of the algorithm in order to support curves of arbitrary degree.
