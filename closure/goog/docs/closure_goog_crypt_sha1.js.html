sha1.js
SHA-1 cryptographic hash. Variable names follow the notation in FIPS PUB 180-3: http://csrc.nist.gov/publications/fips/fips180-3/fips180-3_final.pdf. Usage: var sha1 = new goog.crypt.sha1(); sha1.update(bytes); var hash = sha1.digest();
