singleton.js
This module simplifies testing code which uses many stateful singletons. The goog.testing.singleton.addSingletonGetter function adds a static getInstance method to the class, which in addition to creating and returning a singleton instance, registers it in a global repository. This way the effect of all getInstance calls can be simply reset in tearDown without knowing what singletons were instantiated in the tests. Usage:
  1. Load base.js.
  2. Load goog.testing.singleton. It overrides goog.addSingletonGetter by goog.testing.addSingletonGetter.
  3. Load the code to test.
  4. Call goog.testing.singleton.reset in the tearDown.
