focushandler.js
This event handler allows you to catch focusin and focusout events on descendants. Unlike the "focus" and "blur" events which do not propagate consistently, and therefore must be added to the element that is focused, this allows you to attach one listener to an ancester and you will be notified when the focus state changes of ony of its descendants.
