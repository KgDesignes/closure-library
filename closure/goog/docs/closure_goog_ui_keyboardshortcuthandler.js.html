goog.ui.KeyboardShortcutHandler
Component for handling keyboard shortcuts. A shortcut is registered and bound to a specific identifier. Once the shortcut is triggered an event is fired with the identifier for the shortcut. This allows keyboard shortcuts to be customized without modifying the code that listens for them. Supports keyboard shortcuts triggered by a single key, a stroke stroke (key plus at leat one modifier) and a sequence of keys or strokes.
goog.ui.KeyboardShortcutEvent
Object representing a keyboard shortcut event.
