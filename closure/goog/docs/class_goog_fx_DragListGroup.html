fx.DragListGroup Extends goog.events.EventTarget
A class representing a group of one or more "drag lists" with items that can be dragged within them and between them. Example usage: var dragListGroup = new goog.fx.DragListGroup(); dragListGroup.setDragItemHandleHoverClass(className1); dragListGroup.setDraggerElClass(className2); dragListGroup.addDragList(vertList, goog.fx.DragListDirection.DOWN); dragListGroup.addDragList(horizList, goog.fx.DragListDirection.RIGHT); dragListGroup.init();
