goog.editor.plugins.EnterHandler
Plugin to handle enter keys. This does all the crazy to normalize (as much as is reasonable) what happens when you hit enter. This also handles the special casing of hitting enter in a blockquote. In IE and Safari, the resulting HTML uses one DIV tag per line. In FireFox, the resulting HTML uses BR tags at the end of each line.
